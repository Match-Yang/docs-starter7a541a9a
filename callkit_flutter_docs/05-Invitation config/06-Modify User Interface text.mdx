# Modify User Interface text

Call Kit &#40;ZegoUIKitPrebuiltCallInvitationService&#41;'s UI text provided by the internal components is editable, to modify those, use the <code>innerText</code> config.

Here is the reference code:

```dart
ZegoUIKitPrebuiltCallInvitationService()
    ..innerText.incomingCallPageAcceptButton = "Accept"
    ..innerText.incomingCallPageDeclineButton = "Decline";
```
