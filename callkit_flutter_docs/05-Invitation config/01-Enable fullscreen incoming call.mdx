# Enable fullscreen incoming call

<Warning title="Warning">
**<u>Only offline call invitation supports full screen feature.</u>**
</Warning>

You can change the incoming call dialog from notification to full screen by configuration, as shown belowï¼š

<Frame width="256" height="auto" caption="">
  <img src="https://media-resource.spreading.io/5fa3f99cda659c8c9f2907cbb0242e6c/workspace86/20240223-141756.jpeg" />
</Frame>


The configuration is as follows.

1. set <code>showFullScreen</code> to true in <code>ZegoCallAndroidNotificationConfig</code>

```dart
ZegoUIKitPrebuiltCallInvitationService().init(
    ...
    notificationConfig: ZegoCallInvitationNotificationConfig(
      androidNotificationConfig: ZegoCallAndroidNotificationConfig(
        showFullScreen: true,
      ),
    ),
    ...
  );
```

2. set <code>android:launchMode</code> to <code>singleInstance</code> in <code>AndroidManifest.xml</code>
   * open <code>$&#123;project&#95;root&#125;/android/app/src/main/AndroidManifest.xml</code>
   * find <code>manifest/application/activity</code>

```xml AndroidManifest.xml {5}
<manifest ...>
   ...
   <application ...>
        <activity ...
            android:launchMode="singleInstance"
            ...>
            ...
        </activity>
        ...
    </application>
</manifest>
```
