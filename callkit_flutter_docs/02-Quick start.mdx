## Integrate the SDK


### Add ZegoUIKitPrebuiltCall as dependencies


Run the following code in your project root directory :

```plaintext 
flutter pub add zego_uikit_prebuilt_call
```

### Import the SDK


Now in your Dart code, ​​​​​​​​​​​​​​​​&#8203;import the prebuilt Call Kit SDK.

```dart
​​​​​​​​​​​​​​​​import 'package:zego_uikit_prebuilt_call/zego_uikit_prebuilt_call.dart';
```

### Using the ZegoUIKitPrebuiltCall in your project


* Go to [ZEGOCLOUD Admin Console ](https://console.zegocloud.com/), get the <code>appID</code> and <code>appSign</code> of your project.
* Specify the <code>userID</code> and <code>userName</code> for connecting the Call Kit service.
* Create a <code>callID</code> that represents the call you want to make.
* <code>userID</code> and <code>callID</code> can only contain numbers, letters, and underlines &#40;&#95;&#41;.
* Users that join the call with the same <code>callID</code> can talk to each other.

```dart
class CallPage extends StatelessWidget {
  const CallPage({Key? key, required this.callID}) : super(key: key);
  final String callID;

  @override
  Widget build(BuildContext context) {
    return ZegoUIKitPrebuiltCall(
      appID: yourAppID, // Fill in the appID that you get from ZEGOCLOUD Admin Console.
      appSign: yourAppSign, // Fill in the appSign that you get from ZEGOCLOUD Admin Console.
      userID: 'user_id',
      userName: 'user_name',
      callID: callID,
      // You can also use groupVideo/groupVoice/oneOnOneVoice to make more types of calls.
      config: ZegoUIKitPrebuiltCallConfig.oneOnOneVideoCall(),
    );
  }
}
```

Now, you can make a new call by navigating to this <code>CallPage</code>.

### Android


1. Modify the <code>compileSdkVersion</code> to 33 in the <code>your&#95;project/android/app/build.gradle</code> file.

```xml
minSdkVersion 21
```

3. Modify the <code>kotlin</code> version to &gt;&#61; 1.8.0 and the <code>gradle classpath</code> version to 7.3.0 in the <code>your&#95;project/android/app/build.gradle</code> file:

```xml
   buildscript {
       ext.kotlin_version = '1.8.0'
       repositories {
           google()
           mavenCentral()
       }
   
       dependencies {
           classpath 'com.android.tools.build:gradle:7.3.0'
           classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:$kotlin_version"
   
           // support notification
           classpath 'com.google.gms:google-services:4.3.14'

   }
```

4. Make the <code>gradle</code> version &gt;&#61; 7.0: In the <code>your&#95;project/android/gradle/wrapper/gradle&#45;wrapper.properties</code> file, modify the <code>distributionUrl</code> to <code>https://services.gradle.org/distributions/gradle-7.4-all.zip</code>.



### iOS


1. Add app permissions.

Open the <code>your&#95;project/ios/Podfile</code> file, and add the following to the <code>post&#95;install do |installer|</code> part:

```plaintext
# Start of the permission_handler configuration
target.build_configurations.each do |config|
  config.build_settings['GCC_PREPROCESSOR_DEFINITIONS'] ||= [
    '$(inherited)',
    'PERMISSION_CAMERA=1',
    'PERMISSION_MICROPHONE=1',
  ]
end
# End of the permission_handler configuration
```

```xml
    We require microphone access to connect
```
